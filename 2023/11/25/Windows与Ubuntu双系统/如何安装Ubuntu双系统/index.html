如何安装Ubuntu双系统

前言

由于 WSL 上一些操作还是不方便，本人果断抛弃 WSL 投入 Linux 的怀抱。

本文为本人在 Windows11 系统上安装 Ubuntu22.04 并且配置深度学习环境的过程，可供参考。

首先你需要大约 100G 的空余内存，一块拥有 8G 以上内存的 U盘。

Windows系统：Windows 11
Ubuntu系统：Ubuntu22.04

注意！本文 BIOS 为 UEFI 类型，所以挂载 efi 分区。

参考文章：https://blog.csdn.net/codeHonghu/article/details/111940656

1. 划分磁盘

此步也可以使用 DiskGenius，知道如何使用的人自用。

首先进入计算机管理，找到磁盘管理，在这里可以看到自己电脑的磁盘状态。

我们要做的就是从仍有空间的磁盘中划分出要分给 Ubuntu 系统的空间。

最为重要的是划分出 EFI 分区。

  • 什么是EFI系统分区：https://zhuanlan.zhihu.com/p/262069479

EFI启动是指使用 EFI或UEFI 来启动电脑的一种方式。EFI或UEFI是一种新的固件标准，可以替代传统的BIOS，提供更多的功能和安全性。EFI启动可以从网卡启动，也可以从硬盘或U盘启动。从网卡启动可以实现批量安装系统的功能，称为网刻。

首先我们找到 Windows 的系统盘，如果你没有操作过，那么默认就是 C盘。

右键 C盘，从 C盘 中划分出大约 200MB 的空间，用于 EFI分区。

接下来按照相似操作，从其他盘或者 C盘再划分出约 100G 的空间。（依照你自己想要分配的空间大小分配，给 1T 也是可以的，本人分了256G）

2. 制作 iso U盘镜像文件

2.1 下载 Ubuntu22.04 iso 镜像文件

进入 Ubuntu 官网： https://ubuntu.com/download/desktop

找到对应安装版本，后缀为 .iso

等待下载完成即可。

如果下载速度过慢，可以去清华大学镜像网下载：

清华大学镜像网: https://mirrors.tuna.tsinghua.edu.cn/ubuntu-releases/

2.2 安装 Rufus

我们使用 Rufus 制作 U盘镜像文件。

Rufus 是一款格式化和创建 USB 启动盘的辅助工具。
本软件适用于以下场景：
需要将可引导 ISO (Windows、Linux、UEFI 等) 刻录到 USB 安装媒介的情况
需要处理未安装操作系统的设备的情况
需要在 DOS 环境下刷写 BIOS 或其他固件的情况
需要运行低级工具的情况
Rufus 麻雀虽小，但五脏俱全！

官方网站： https://rufus.ie/zh/

根据自己的 Windows 系统架构安装对应版本。

2.3 制作 U盘镜像

插入 U盘，打开 Rufus，在此界面，设备选项应该会读取到插入的 U盘，检查一下名称，确认无误。

图 2.1

注意 进行镜像文件创造会格式化 U盘！

引导文件选择刚刚下载的 Ubuntu22.04 iso 文件，其他默认，点击开始即可等待镜像文件创建。

3. 安装 Ubuntu22.04 系统

重启电脑，保证 U盘插上，进入 bios 界面，将启动选项第一项改为从 U盘启动。

进入 bios 界面修改启动项：https://zhuanlan.zhihu.com/p/73403185

然后电脑会自动重启，会出现一个带字的黑框，用方向键操控，选择 “Try to install ubuntu”，回车确认，就进入了 Ubuntu 安装界面。

前面语言随便默认即可，当进入如图界面时，选择 其他选项：

图 3.1

进入到这样的界面：

图 3.1

点击空闲的区域就会弹出如下界面：

图 3.2

接下来按照给出的信息找到自己分出的空余空间，进行 Ubuntu 系统的分盘：

3.1 EFI分区

EFI 分区需要选取之前在 Window C盘中分出的 300MB 的空余分区！

  • 大小：300 MB
  • 分区类型：逻辑分区
  • 新分区位置：空间起始位置
  • 用于：EFI 系统文件

3.2 swap 交换分区

这是 Linux 也就是 Ubuntu 的交换区目录，大小一般为内存的 2 倍，在电脑内存不足时，系统调用这片区域，来运行程序。可以将其分为8～16G。

  • 大小：8～16G
  • 分区类型：逻辑分区
  • 新分区位置：空间起始位置
  • 用于：交换分区

3.3 / 根目录

这是 Linux 也就是 Ubuntu 的根目录，就是一个反斜杠 /，相当于Windows的C盘，ubuntu装软件，默认装在这里。

  • 大小：40G起步，空间大的可以多给一些（本人给了100G）
  • 分区类型：逻辑分区
  • 新分区位置：空间起始位置
  • 用于： ext4
  • 挂载点：/

3.4 /home 分区

这是 ubuntu 的主分区，相当于windows的其他盘。

  • 大小：剩下的空间全给他
  • 分区类型：主分区
  • 新分区位置：空间起始位置
  • 用于： ext4
  • 挂载点：/home

至此，ubuntu 系统的分区任务以及完成，后面一直默认点下去等待安装就可以了。

第一次安装完毕可能关不了机，可以强制重启，记得拔下 U盘，进入这个界面并且顺利进入 Ubuntu 说明安装成功～
