New Mac Configuration

Preface

Here is a memorandum for my personal reference, which may also be beneficial for first-time Mac users.

1 Basic Settings

1.1 Turn off the startup sound

Unclick System Settings -> Sound -> Play sound on startup

2 Useful Tools

The following links are to some superb developers.

Better365 official web: https://www.better365.cn/

2.1 iShot

This may be the best free snipping tool that supports marking, adding text, OCR, and capturing scrolling screenshots, among other features.

Article: https://sspai.com/post/59416#!

Figure-2.1 iShot

You can find it on the App Store.

2.2 iBar

iBar is a utility used to beautify your Mac’s menu bar by hiding unnecessary icons for a cleaner look.

Figure-2.2 iBar

You can find it on the App Store.

2.3 iRightMouse

iRightMouse is a tool that enhances the functionality of your right-click menu, allowing for quick actions such as creating new files, opening the current folder in the terminal (which is very useful!), and copying the file path, among other features.

Figure-2.3 iRightMouse

You can find it on the App Store.

2.4 FastZip

FastZip extends your compression capabilities, adds support for additional compression formats such as RAR and 7z.

Figure-2.4 FastZip

You can find it on the App Store.

2.5 Better Menubar

This tool displays your CPU, RAM and Network status and also provides icons for quick access to tasks.

Figure-2.5 BetterMenubar

You can find it on the App Store.

2.6 BetterAndBetter

This might be the tool I use most frequently. It offers more gesture operations, more keyboard shortcuts, and even allows you to write your own code to execute shortcuts. When used in conjunction with the Shortcuts app, it becomes incredibly “cyberpunk”.

Figure-2.6 BetterMenubar

Link: https://www.better365.cn/bab2.html

3 About network

3.1 Tailscale

Tailscale is a VPN service that enables encrypted point-to-point connections using the open source WireGuard protocol.

Figure-3.1 Tailscale

Link: https://tailscale.com/

3.2 Shadowrocket

Shadowrocket is a VPN tool.

Figure-3.2 Shadowrocket

You can find it on the App Store.

3.3 Electerm

Electerm is an open-sourced terminal/ssh/telnet/serialport/RDP/VNC/sftp client(linux, mac, win).

Figure-3.3 Electerm

github link :https://github.com/electerm/electerm

3.4 FinalShell

This is a free and compact software for SSH connections, offering a variety of tools.

Figure-3.4 FinalShell

Link: https://www.hostbuf.com/

Tip: You need to allow your Mac to download software from the web.

4 About design

4.1 Figma

Figma is the leading collaborative design tool for building meaningful products. No need to say more.

Figure-4.1 Figma

4.2 Spline

Spline is a free 3D design software with real-time collaboration to create web interactive experiences in the browser.

Figure-4.1 Spline
